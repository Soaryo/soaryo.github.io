变量提升

Javascript执行过程进入新函数时，函数内声明的所有变量都会提升到函数顶部。
被提升的只有变量的声明。

var f = function(a){return a};

  1. 匿名函数作为参数传递给其他函数，接收函数利用匿名函数完成某些事情
  2. 定义匿名函数完成一次性任务

传递对象

拷贝对象或将对象传递给函数时，传递的是该函数的引用，在引用上的改动，会影响它所引用的原对象

内建对象

  1. 数据封装类对象:Object,Array,Boolean,Number,String;
  2. 工具类对象:Math,Date,RegExp;
  3. 错误类对象;

内建对象:Function

prototype属性

  1. 每个函数的prototype属性都指向一个对象
  2. 只有在该函数是构造器的时发挥作用
  3. 通过该函数创建的所有对象都会持有一个该prototype属性的引用，并可以将其当做自身属性使用。

call() and apply()

原型(prototype)

原型属性

  1. 函数定义时被创建的属性，prototype属性初始值为一个空对象
  2. 原型中定义的属性和方法，只有当构造器函数时才有用
  3. 原型具有“实时（live）性”
  4. __proto__是实例对象的属性，prototype是构造器函数属性

原型陷阱

  1. 原型对象完全替换时，可能会触发原型链的异常
  2. prototype.constructor会改变
